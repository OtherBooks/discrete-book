Without any restriction, there would be \({19\choose 12}\) ways to distribute the stars. Now we must use PIE to eliminate all distributions in which one or more student gets more than one star: \begin{equation*} {19 \choose 12} - \left[{13 \choose 1}{17 \choose 12} - {13\choose 2}{15 \choose 12} + {13\choose 3}{13 \choose 12}\right] = 1716. \end{equation*}
