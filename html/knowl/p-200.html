{Cartesian Product} Given sets \(A\) and \(B\), we can form the set \(A \times B = \{(x,y) \st x \in A \wedge y \in B\}\) to be the set of all ordered pairs \((x,y)\) where \(x\) is an element of \(A\) and \(y\) is an element of \(B\). We call \(A \times B\) the Cartesian product of \(A\) and \(B\).
