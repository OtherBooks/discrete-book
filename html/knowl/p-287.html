In the expansion, there will be only one \(x^5\) term and one \(y^5\) term. This is because to get an \(x^5\), we need to use the \(x\) term in each of the copies of the binomial \((x+y)\), and similarly for \(y^5\). What about \(x^4y\)? To get terms like this, we need to use four \(x\)'s and one \(y\), so we need exactly one of the five binomials to contribute a \(y\). There are 5 choices for this, so there are 5 ways to get \(x^4y\), so the coefficient of \(x^4y\) is 5. This is also the coefficient for \(xy^4\) for the same (but opposite) reason: there are 5 ways to pick which of the 5 binomials contribute the single \(x\). So far we have \begin{equation*} (x+y)^5 = x^5 + 5x^4y + \underline{~?~}~x^3y^2 + \underline{~?~}~x^2y^3 + 5 xy^4 + y^5. \end{equation*}
