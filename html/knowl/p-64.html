This is the set of all elements which are both elements of \(A\) and not elements of \(B\). What have we done? We've started with \(A\) and removed all of the elements which were in \(B\). Another way to write this is the set difference: \begin{equation*} A \cap \bar B = A \gls{setminus} B. \end{equation*}
