Solution3.8.1.1.7.d.1

First compute the number of routes to the donut shoppe avoiding the pothole: \begin{equation*} {16 \choose 7} - {7 \choose 3}{8 \choose 6} \end{equation*}

Then you still need to go to work from there. Thus the answer is: \begin{equation*} \left({16 \choose 7} - {7 \choose 3}{8 \choose 6}\right){6 \choose 2} \end{equation*}
