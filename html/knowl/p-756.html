Note that now we can guess a closed formula for the \(n\)th term of the sequence: \(a_n = n+3\). To be sure this will always work, we could plug in this formula into the recurrence relation: \begin{equation*} 2a_{n-1} - a_{n-2} = 2((n-1) + 3) - ((n-2) + 3) = 2n + 4 - n - 1 = n + 3 = a_n. \end{equation*}
