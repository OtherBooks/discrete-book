Definition2.2.3

{Special sets}

\gls{emptyset} The empty set is the set which contains no elements.
[1ex] \glsU The universe set is the set of all elements.
[1ex] \glsN The set of natural numbers. That is, \(\N = \{0, 1, 2, 3\ldots\}\).
[1ex] \glsZ The set of integers. That is, \(\Z = \{\ldots, -2, -1, 0, 1, 2, 3, \ldots\}\).
[1ex] \glsQ The set of rational numbers.
[1ex] \glsR The set of real numbers.
[1ex] \gls{PowA} The power set of any set \(A\) is the set of all subsets of \(A\).
