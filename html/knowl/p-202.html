Let \(A = \{a_1,a_2, a_3, \ldots, a_m\}\) and \(B = \{b_1,b_2, b_3, \ldots, b_n\}\) (so \(|A| = m\) and \(|B| = n\)). The set \(A \times B\) contains all pairs with the first half of the pair being \(a_i\) for some \(i\) and the second being \(b_j\) for some \(j\). In other words: \begin{align*} A \times B = \{ \amp (a_1, b_1), (a_1, b_2), (a_1, b_3), \ldots (a_1, b_n),\\ \amp (a_2, b_1), (a_2, b_2), (a_2, b_3), \ldots, (a_2, b_n),\\ \amp (a_3, b_1), (a_3, b_2), (a_3, b_3), \ldots, (a_3, b_n),\\ \amp \vdots\\ \amp (a_m, b_1), (a_m, b_2), (a_m, b_3), \ldots, (a_m, b_n)\}. \end{align*}
