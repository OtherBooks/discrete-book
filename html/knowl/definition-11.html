Definition3.2.1

{Bit Strings}

  • A \(n\)-bit string is a bit string of length \(n\). That is, it is a string containing \(n\) symbols, each of which is a bit, either 0 or 1.
  • The weight of a bit string is the number of 1's in it.
  • \gls{Bn} is the set of all \(n\)-bit strings.
  • \gls{Bnk} is the set of all \(n\)-bit strings of weight \(k\).
