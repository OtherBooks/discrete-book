If a set is finite, then we can describe it by simply listing the elements. Infinite sets exists though, so we need to be able to describe them as well. For instance, if we want \(A\) to be the set of all even natural numbers, would could write, \begin{equation*} A = \{0, 2, 4, 6, \ldots\}, \end{equation*} but this is a little imprecise. Better would be \begin{equation*} A = \{x \in \N \st \exists n\in \N ( x = 2 n)\}. \end{equation*}
