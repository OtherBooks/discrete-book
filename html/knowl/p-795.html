How do we know this? For the recursive definition, we need to specify \(a_0\). Then we need to express \(a_n\) in terms of \(a_{n-1}\). If we call the first term \(a\), then \(a_0 = a\). For the recurrence relation, by the definition of an arithmetic sequence, the difference between successive terms is some constant, say \(d\). So \(a_n - a_{n-1} = d\), or in other words, \begin{equation*} a_0 = a \qquad a_n=a_{n-1}+d. \end{equation*}
